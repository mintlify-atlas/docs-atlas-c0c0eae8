Apache Hadoop is a battle-tested framework that enables distributed storage and processing of massive datasets — from gigabytes to petabytes — across clusters of thousands of commodity machines. Built around four core subsystems (HDFS, YARN, MapReduce, and Common), Hadoop provides the reliability, scalability, and fault tolerance required for enterprise data infrastructure.

## What you can build

Hadoop powers large-scale data pipelines, analytics workloads, and storage backends across industries. Common use cases include:

- **Batch data processing** — Run MapReduce or Spark jobs over petabytes of log, clickstream, or sensor data stored in HDFS
- **Data lake storage** — Store structured and unstructured data in HDFS as a central repository for downstream analytics tools
- **ETL pipelines** — Use DistCp to move data between clusters or cloud storage, and Hadoop Streaming to process data with any language
- **Multi-tenant analytics** — Isolate workloads using YARN Capacity Scheduler queues with guaranteed resource allocations
